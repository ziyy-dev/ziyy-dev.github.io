# `<a>`

The `<a>` element (or anchor element), with its href attribute, creates a hyperlink on the terminal to a URL.

Content within each `<a>` should indicate the link's destination.

## Example

```html
<a href="https://example.com">Visit Example</a>
```

## Attributes

This element includes the [global attributes](../globals.mdx).

### `href`

The URL that the hyperlink points to.
