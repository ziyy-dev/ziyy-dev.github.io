# `<b>`

The `<b>` element is used to style text as bold. It is often used to emphasize important text or to make certain words stand out in a terminal output.

## Example

```ziyy
<b>This text is bold
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<strong>`

## CSS class

You can use `b` CSS class to apply bold style to an element.
