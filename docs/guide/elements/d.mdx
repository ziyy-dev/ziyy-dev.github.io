# `<d>`

The `<d>` element is used to style text as dim. It is often used to indicate less important information or to de-emphasize certain parts of the text in terminal output.

## Example

```html
<d>This text is dim</d>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<dim>`

## CSS class

You can use `d` CSS class to apply dim style to an element.
