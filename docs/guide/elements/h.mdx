# \<h>

The `<h>` element is used hide text in the terminal. It is often used to hide sensitive information or to prevent certain text from being displayed in the terminal output.

## Example

```ziyy
<h>This text is hidden</h>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<hide>`

## CSS class

## CSS class

You can use `h` CSS class to hide an element.
