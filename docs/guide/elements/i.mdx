# \<i>

The `<i>` element is used to style text as italic. It is often used to indicate a different voice or to emphasize certain words in a terminal output.

## Example

```ziyy
<i>This text is italic</i>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<em>`

## CSS class

You can use `i` CSS class to apply italic style to an element.
