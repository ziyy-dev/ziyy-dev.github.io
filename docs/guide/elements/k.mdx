# `<k>`

The `<k>` element is used to make text blink in the terminal. It is often used to draw attention to specific text or to indicate that something requires immediate attention.

## Example

```html
<k>This text is blinking</k>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<blink>`

## CSS class

You can use `k` CSS class to apply blink style to an element.
