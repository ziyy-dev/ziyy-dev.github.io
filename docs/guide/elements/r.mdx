# \<r>

The `<r>` element is used to invert the foreground and background colors of text in the terminal. It is often used to create a highlight effect or to draw attention to specific text.

## Example

```ziyy
<r>This text has inverted colors</r>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<invert>`

## CSS class

You can use `r` CSS class to invert foreground and background colors of an element.
