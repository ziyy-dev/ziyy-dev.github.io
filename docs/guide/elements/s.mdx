# \<s>

The `<s>` element is used to style text as strikethrough. It is often used to indicate that certain text is no longer relevant or has been removed.

## Example

```ziyy
<s>This text is strikethrough</s>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<del>`

## CSS class

You can use `s` CSS class to apply strikethrough style to an element.
