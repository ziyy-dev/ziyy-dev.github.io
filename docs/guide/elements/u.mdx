# `<u>`

The `<u>` element is used to style text as underlined. It is often used to emphasize important text or to make certain words stand out in a terminal output.

## Example

```html
<u>This text is underlined</u>
```

## Attributes

This element includes the [global attributes](../globals.mdx).

### double

Applies a double underline instead of a single underline to the text.

## Alias

- `<ins>`

## CSS class

You can use `u` CSS class to apply underline style to an element.
