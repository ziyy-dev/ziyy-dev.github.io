# \<uu>

The `<uu>` element is used to style text as double underline. It is often used to indicate a strong emphasis or to highlight important information in terminal output.

## Example

```ziyy
<uu>This text is double underlined</uu>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## CSS class

You can use `uu` CSS class to apply double underline style to an element.
