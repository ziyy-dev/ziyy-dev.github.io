# \<x>

The `<x>` element is used to style background of text with a specific color. It is often used to highlight important information or to make certain text stand out in terminal output.

## Example

```ziyy
<x red>This text has a red background</x>
```

## Attributes

This element only includes the [global attributes](../globals.mdx).

## Alias

- `<bg>`
