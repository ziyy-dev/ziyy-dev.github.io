# \<ziyy>

The `<ziyy>` element represents the root (top-level element) of an Ziyy document, so it is also referred to as the root element. All other elements must be descendants of this element. There can be only one `<ziyy>` element in a document.

## Examplae

```ziyy
<ziyy>
  <p>A paragraph of text</p>
  <p>Another paragraph of text</p>
</ziyy>
```

## Attributes

This element includes the [global attributes](../globals.mdx).
