# Introduction
Ziyy is a tool for styling terminals using HTML-like tags. It allows you to create visually appealing terminal outputs with ease, leveraging familiar HTML syntax to define styles and layouts.

## Why Ziyy?
Ziyy is designed to simplify terminal styling by providing a straightforward way to apply styles using HTML-like tags. It eliminates the need for complex terminal escape codes, making it accessible for developers who are comfortable with HTML.

## Key Features
- **HTML-like Syntax**: Use familiar HTML tags to style terminal outputs.
- **Easy to Use**: Simple API for applying styles to terminal text.
- **Cross-Platform**: Works on any platform that supports terminal output.
